12 Sites in 12 Weeks

I've always wanted to learn how to code. I might take classes but I was inspired by 180 lady So I'm challenging myself to DO it. To make 12 sites with different elements that give me a good foundation on HTML/CSS, a little java, and if I'm feeling Groovy, maybe a little Ruby.


Guidelines

There are several guidelines for this project that I have to follow to make sure I'm pushing myself to learn

  1. Each site must be shipped on Sunday by midnight
  2. Each site must include 5 new elements that weren't present in the last site
  3. The code for each site must be shared via Github
  4. I can record resources, questions, and lessons learned in the blog
  5. No templates the first 3 weeks.All code done by hand
