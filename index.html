12Sites of Xmas

The Gift of Code

Christmas is in 12 weeks and this year I've decided to give myself the gift of code. I've always wanted to learn how to build websites. There are a ton of MOOCs and classes out there doing a great job of teaching web design skills but I believe the best way to learn is getting your hands dirty and building it yourself. Inspired by Jennifer DeWalt I'm challenging myself to learn by building. Follow along here and be the first to get my new site each Monday morning here .

Guidelines

Here are the rules to the game

  • Each site must be shipped on Sunday by midnight
  • Each site must include 5 new elements that weren't present in the last site
  • The code for each site must be shared via Github
  • I can record resources, questions, and lessons learned in the blog
  • No templates the first 3 weeks.All code done by hand
